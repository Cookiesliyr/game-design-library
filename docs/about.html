What is the Game Design Library?

The Game Design Library is a curated catalogue of game-design links. We created this site with one goal: to make it easy for game-designers and non-designers alike find relevant, useful information about game design, and their specific game-design problems.

We scoured the internet to find relevant pages on the theory and practice of game design; everything from general high-level concepts such as player types, to genre- and problem-specific pages (e.g. how can I design interesting boss battles for my JRPG?

Note that GDL is about game design, not game development. We don't list any links about programming, game art, character rigging, marketing tips, monetization, or any other facets of game development; we focus strictly on game design.

The GDL is an ongoing project - we add links on an ongoing basis. Contributions are always welcome! If you are interested, please take a look at our data.json file, click edit (pencil icon), make your changes, and then open a PR so we can review/integrate them.
